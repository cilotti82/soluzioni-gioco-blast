# Block Blast Solver

App web statica per risolvere schermate Block Blast 8x8 da telefono.

## Uso

1. Apri il sito GitHub Pages.
2. Carica lo screenshot del gioco.
3. Premi `Auto` e controlla griglia e pezzi.
4. Correggi eventuali celle cliccandole.
5. Premi `Risolvi`.

Coordinate: `R1 C1` e' la cella in alto a sinistra.

Link Pages previsto: https://cilotti82.github.io/soluzioni-gioco-blast/
